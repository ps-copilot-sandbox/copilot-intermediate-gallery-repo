# Guías de Demos para Galería de Fotos y Portafolio

Esta carpeta contiene guías de demos paso a paso para aprender y practicar las funcionalidades de GitHub Copilot en la aplicación de Galería de Fotos y Portafolio. Cada demo se construye sobre el anterior, ayudándote a dominar las capacidades de Copilot en un proyecto Next.js del mundo real.

## Secuencia de Demos y Descripciones

### 1. Demo de Funcionalidades ([features-demo.md](features-demo.md))
**Descripción:**  
Comienza aquí para explorar las funcionalidades principales de Copilot. Aprende a usar comandos de chat, generar código y revisar sugerencias de IA.  
**Habilidades Clave:**  
- Descubrir comandos disponibles de Copilot  
- Obtener resúmenes de proyecto y explicaciones de código  
- Generar y revisar completado de código  
- Hacer commit de cambios con Copilot

---

### 2. Component Development with Copilot Demo ([component-development.md](component-development.md))
**Descripción:**  
Aprende a aprovechar Copilot para el desarrollo de componentes React en el mundo real. Construye componentes TypeScript, mejora la seguridad de tipos, refactoriza código, agrega animaciones y genera documentación.  
**Habilidades Clave:**  
- Generar componentes React con TypeScript
- Crear y usar tipos compartidos
- Refactorizar para rendimiento con useMemo
- Agregar animaciones con Framer Motion
- Generar documentación JSDoc y pruebas

---

### 3. Demo de Prácticas de Ingeniería ([engineering-practices.md](engineering-practices.md))
**Descripción:**  
Profundiza en herramientas profesionales de Copilot para equipos. Inspecciona el proceso de decisión de Copilot, comparte conversaciones de chat y explora prompts del sistema para generación de código consistente.  
**Habilidades Clave:**  
- Depurar sugerencias de Copilot  
- Exportar/importar conversaciones de chat  
- Comprender y administrar prompts del sistema  
- Colaborar usando conversaciones compartidas

---

### 4. Demo de Personalización de Copilot ([customize-copilot.md](customize-copilot.md))
**Descripción:**  
Aprende técnicas avanzadas de personalización. Monitorea uso premium, cambia entre modelos, usa archivos de prompts, experimenta con modos de agente personalizados y configura instrucciones personalizadas para tu equipo.  
**Habilidades Clave:**  
- Rastrear uso premium de Copilot  
- Cambiar entre modelos de IA  
- Crear y usar archivos de prompts  
- Utilizar modos de agente personalizados, instrucciones personalizadas y servidores MCP

---

### 5. Demo de Copilot Spaces ([copilot-spaces.md](copilot-spaces.md))
**Descripción:**  
Colabora en Copilot Spaces dedicados. Crea un Space, establece objetivos, agrega archivos de contexto y trabaja en conjunto para implementar nuevas funcionalidades con asistencia de IA.  
**Habilidades Clave:**  
- Crear y administrar Copilot Spaces  
- Establecer objetivos de desarrollo  
- Colaborar y compartir progreso  
- Implementar y probar nuevas funcionalidades

---

### 6. Demo de Agente en la Nube ([coding-agent.md](coding-agent.md))
**Descripción:**  
Experimenta GitHub Copilot como agente en la nube para acelerar la construcción y mejora de funcionalidades en tu aplicación de Galería de Fotos y Portafolio.  
**Habilidades Clave:**  
- Asignar Copilot a issues de GitHub  
- Revisar pull requests generados por Copilot y detalles de sesión  
- Practicar revisión de código colaborativa e iteración  

---

### 7. Demo de GitHub MCP: de Issue a Pull Request ([github-mcp-issue-to-pr-demo.md](github-mcp-issue-to-pr-demo.md))
**Descripción:**  
Aprende a usar el servidor GitHub MCP desde VS Code para que Copilot cree trabajo real en GitHub y lo conecte con un flujo de implementación asistido. Esta demo enseña cómo pasar de un prompt a un issue, y de ahí a un PR revisable.  
**Habilidades Clave:**  
- Verificar la configuración y autenticación del servidor GitHub MCP
- Crear issues de GitHub desde Copilot Chat usando herramientas MCP
- Refinar alcance y criterios de aceptación con prompts iterativos
- Conectar el issue con GitHub Copilot para continuar hacia un PR
- Revisar el resultado con criterio técnico

---

## Cómo Usar Estos Demos

1. Comienza con el primer demo y trabaja a través de cada guía en orden.
2. Sigue las instrucciones y prompts en cada archivo.
3. Marca las listas de verificación de completitud mientras progresas.
4. Comparte tus aprendizajes y resultados con tu equipo.
