# Cloud Agent Demo

---

## What You'll Learn
By the end of this demo, you will:
- [ ] Use `/chronicle standup` to generate a 24-hour activity report
- [ ] Create a feature issue using Copilot's `/create-issue` slash command
- [ ] Review Copilot-generated pull requests and session details

**Estimated Time:** 10 minutes

## 🎯 Step 1: Generate a Standup Report with `/chronicle standup`

1. Go to the GitHub repository for the Photo Gallery & Portfolio application.
2. Select the **Agents** tab in the repository navigation bar.
3. It will open the Agents page located [https://github.com/ps-copilot-sandbox/copilot-intermediate-gallery-repo/agents](https://github.com/ps-copilot-sandbox/copilot-intermediate-gallery-repo/agents).
4. Type the slash command:

```text
/chronicle standup --days 5
```

5. Review the generated report for work completed in the last 5 days.
6. For an extra challenge, look through the other slash commands available in the Copilot panel and try them out.

## 🛠️ Step 2: Create a Feature Issue via Copilot Slash Commands

1. Confirm you are in the GitHub repository for the Photo Gallery & Portfolio application.
2. Click the Copilot logo in the top-right next to the search bar.
3. Look through the available slash commands in the Copilot panel by typing `/` and scrolling through the list.
4. Use the following command to draft a new feature issue:

```markdown
/create-issue Add photo download support to the gallery app. Keep the implementation modular, TypeScript-friendly, and consistent with the existing UI. Include loading and error states, use async/await for I/O, avoid hardcoded secrets or URLs, and follow existing project conventions.
```

5. Look over the issue draft generated by Copilot
6. Assign the issue to **Copilot**, **Claude**, or **Codex** to implement the feature.

## 📚 Step 3: Review Copilot's Changes

1. Scroll to the _Selected Agent_ `linked a pull request that will close this issue` and click the link to view the PR.
2. Review the session details in the [agents tab](https://github.com/copilot-co/copilot-intermediate-gallery-repo/agents) to see how Copilot implemented the feature.
3. Go to the PR and review the file changes made by Copilot.

_If time permits, open the PR in GitHub Copilot app and test out the changes made by Copilot._
